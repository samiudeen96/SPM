Value Waterfall Builder

A waterfall chart can be a useful tool to summarize the value that you bring to a potential customer. This tool will allow you to build your own simple waterfall chart in a few minutes. 

Using This Tool

You can include up to 5 elements in your waterfall of 3 different types:

  • Next Best Alternative - the option that your customer will choose if they don't purchase your solution
  • Positive Value Drivers - differentiating capabilities that provide value to the customer
  • Negative Value Drivers - instances where your solution provides inferior value to the customer relative to the competition

Please note:

  • Please enter numbers only (you can enter in a decimal)
  • If a value is supposed to be negative, it must be entered as a negative number to appear correctly in the waterfall
